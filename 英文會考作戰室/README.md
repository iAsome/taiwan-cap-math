# 會考英文作戰室

可攜式離線網站。整個資料夾可複製到隨身碟、雲端硬碟或壓縮後分享，不需要安裝套件或登入。

## 範圍界定（先讀這裡，避免誤會）

國中教育會考英語科分「閱讀」與「聽力」兩節：閱讀 40–45 題／60 分鐘（單題語彙語法＋題組克漏字／整段理解），聽力 20–30 題／25 分鐘，成績依 80%／20% 加權合併計算。因此：

- 本站模擬考、單元小考只涵蓋**閱讀**與**字彙語法**，聽力題型以**文字腳本＋情境描述**呈現，**不含語音檔**，練習的是聽力常考的對話情境判斷、而非真正的聽力測驗，請勿與官方聽力測驗混淆。
- 教育部歷屆試題僅公開閱讀題本與逐題答案，聽力題本的逐字稿與音檔並未隨題本公開，因此「考卷館」（歷屆逐題轉錄）**只收錄閱讀部分**；少數依賴圖片、圖表或版面（例如地圖、月曆、對話框版面）而無法轉錄為等義文字的題目，已在各年度說明中列出並略過。
- 講義依 12 年國教英語課綱的年級文法脈絡（時態、詞類、句型）與部定 2000 字基本字彙表分冊，而非中文科的「課文單元」模式。

## 內容

- 5 大領域、20 個英文單元全冊講義（文法規則／核心字彙、易混淆點釐清、例句與句型公式、推導說明、判斷流程、易錯提醒、單元小測）與完成進度（保存在瀏覽器本機）。
- 小考題庫：20 個單元小考＋5 份領域總複習，全部由 `quiz-taxonomy.js` 的小節 template 生成，模考與小考共用同一份題目來源。
- 每份小考都有專屬 `quiz-banks/<小考ID>/` 題庫維護資料夾，標明小節分類與來源。
- 自訂「英文範圍模擬考」：涵蓋全部 20 單元的閱讀／字彙語法混合模考，聽力題型以文字腳本呈現。
- 「我的考卷」會保存本機瀏覽器考過的卷子，可回看當次題目與詳解。
- 106–115 年官方英語科題本、答案、試題說明存於 `official-data`，可離線開啟；「考卷館」收錄逐題轉錄的閱讀題目與詳解。
- 十年分析頁：英文題數統計、5 大領域分布、年度逐題主概念帳本。
- 三層資料優先規則：教育部／會考官方資料 > 臺灣出版社 > 經驗證的網路資料。
- 技巧與常見說法審核庫，標示「通過／有條件／不採用」及驗證理由。

## 使用

1. 保留資料夾內所有檔案與子資料夾，不要只傳送 `index.html`。
2. 雙擊 `開啟會考英文作戰室.cmd`（如有提供）；也可直接開啟 `index.html`。
3. 建議分享時將整個資料夾壓縮成 ZIP，接收者解壓縮後再使用。

網站、講義、題庫、詳解與官方 PDF 均可離線使用；外部來源網站連結需要網路。

學習進度記錄在每位使用者自己的瀏覽器中，不會跟著資料夾傳給別人。

生成題是原創練習，並非官方題目或預測題；正式規格、答案與勘誤以國中教育會考官方網站為準。
