# 會考國文作戰室

依台灣國中教育會考規格製作的國文全範圍講義、模擬考與近十年歷屆考卷網站，架構比照 [taiwan-cap-math](https://github.com/iAsome/taiwan-cap-math)（會考數學作戰室）製作。

公開網址：https://iasome.github.io/taiwan-cap-chinese/

## 功能

- **全範圍模擬考**：42 題四選一、70 分鐘，依現行官方題本結構與五大內容類別組卷；同一卷別種子可重現同一份試卷。
- **28 單元全冊講義**：涵蓋國一到國三，字音字形、詞語運用、語文常識、文本統整、句段篇章理解賞析五大類別，每單元含核心觀念、重點語料與判斷方法、標準判讀流程、快解技巧與 30 秒小測。
- **單元小考題庫**：依年級 × 學期切分，另有上／下學期與全年總複習。
- **題型與技巧地圖**：常見題型對照判斷方法與最常見的失分點。
- **逐題分析**：以官方題本為母體，逐題完成主概念編碼；目前完整轉錄年度為 115 年，106–114 年題本與答案已下載存檔、轉錄中。
- **資料與技巧審核**：三層資料優先順序（官方 → 出版社 → 公開網路技巧），公開技巧逐條標示通過／有條件／不採用。
- **近十年考卷館**：連結官方歷屆題本；115 年可直接在站內作答並看逐題詳解，其餘年度陸續開放。
- **我的考卷**：本機瀏覽器保存作答紀錄，可回頭查看當次題目與詳解。

## 專案結構

```
data.js            國文全冊 28 單元講義內容、五大類別、技巧審核庫
questions.js        模擬考／小考題目生成引擎（題庫變體池＋種子亂數＋洗牌選項）
archive-exams.js    歷屆會考題本逐題轉錄（含正確答案、詳解、技巧、易錯點）
analysis-data.js    逐題主概念編碼帳本（僅收錄已完整轉錄核對的年度）
app.js               前端邏輯：畫面切換、模考流程、計時、批改、講義與分析頁渲染
index.html / styles.css   頁面結構與樣式
official-data/       官方歷屆題本、答案與試題說明 PDF
```

## 資料來源與限制

- 命題範圍依十二年國教語文領域（國語文）課程綱要第四學習階段；考試結構依國中教育會考官方網站公告。
- 官方僅公布各年度整體題本與參考答案，未逐題公布單元標籤；因此「每題主概念」是本站依題目判斷核心建立的研究編碼，並標明尚未核對的年度，不假造未經核實的統計。
- 生成題為原創練習，非官方預測題；正式資訊以會考官方公告為準。

## 開發

純靜態網站，無建置流程；用任一本機伺服器開啟 `index.html` 即可（例如 `npx serve .`）。
