# 會考生物作戰室

可攜式離線網站，架構與功能比照 [會考數學作戰室](../數學會考作戰室/README.md)：整個資料夾可複製、離線開啟，不需要安裝套件或登入。

## 範圍界定（先讀這裡，避免誤會）

國中教育會考沒有獨立的「生物」考科。生物是自然科（生物／理化／地球科學）的一部分，自然科全部都是四選一選擇題，沒有非選擇題。因此：

- 講義、小考題庫只涵蓋生物這個學習領域，不含理化、地球科學。
- 模擬考採用本站自訂規格（30 題四選一、30 分鐘），**不是**官方獨立考科規格；頁面上會清楚聲明。
- 歷屆十年題目，是從 106–115 年官方「自然科」題本中，逐題人工比對、篩出屬於生物的題目後轉錄整理而成；圖表題若無法用文字完整還原，會在題幹中補述表格數據或直接略過不收錄。

## 內容

- 3 大主題、15 個國中生物單元全冊講義（核心觀念、觀念澄清、公式或關鍵事實、解題流程、快解技巧、30 秒觀念測驗），依南一／翰林／康軒三版本共同的章節脈絡整理，涵蓋「生物體的構造與功能」「生命的延續與生物多樣性」「生態系中的生物與環境」。
- 18 份小考題庫：3 份年級總複習＋15 個單元小考；其中細胞、遺傳、生態等重點單元另有 `quiz-taxonomy.js` 細分類種子出題，其餘單元 fallback 到單元層級程序化出題。全部為四選一選擇題，交卷即看逐題詳解、技巧與易錯點。
- 「生物模擬測驗」涵蓋全部 15 個單元，30 題、30 分鐘，是原創練習，不是官方題本節錄。
- 「我的考卷」保存本機瀏覽器考過的卷子，可回看當次題目與詳解。
- 106–115 年官方自然科題本、答案存於 `official-data/`，可離線查閱原始 PDF。
- 考卷館與十年分析頁：106–115 年共 10 屆自然科題本中挑出的生物題已全數轉錄核對完成（`archive-exams.js`），可直接作答並看詳解；十年分析頁統計主題分布、單元帳本與逐年逐題編碼。
- 三層資料優先規則：教育部／會考官方資料 > 臺灣出版社 > 經驗證的網路資料，並逐條標示常見說法「通過／有條件／不採用」。

## 使用

1. 保留資料夾內所有檔案與子資料夾，不要只傳送 `index.html`。
2. 直接開啟 `index.html` 即可使用。
3. 建議分享時將整個資料夾壓縮成 ZIP，接收者解壓縮後再使用。

## 開發者檢查

`tools/verify-data.mjs` 是本網站的自我檢查腳本，驗證單元數、模考與小考題數、`quiz-taxonomy.js` 對應關係，以及 `archive-exams.js` 與 `analysis-data.js` 逐題 unitId 是否一致：

```bash
node tools/verify-data.mjs
```

學習進度記錄在每位使用者自己的瀏覽器中，不會跟著資料夾傳給別人。

生成題是原創練習，並非官方題目或預測題；考卷館題目雖轉錄自官方自然科題本，但正式規格、範圍與占分仍以國中教育會考官方網站為準。
