# 理化會考作戰室

可攜式離線網站，架構與功能比照 [會考數學作戰室](../README.md)：整個資料夾可複製、離線開啟，不需要安裝套件或登入。

## 內容

- 21 個國中理化單元（國二上 7 單元、國二下 6 單元、國三上 4 單元、國三下 4 單元），依康軒／翰林／南一三版本共同的章節順序整理，涵蓋物質、反應、運動與力、能量、波動、電磁六大領域。**理化橫跨國二、國三共 4 個學期，不是只有國二**：化學相關內容（化學反應、氧化還原、酸鹼、反應速率、有機化合物）集中在國二下學期教完；國三則整年都是物理（力學、電學、磁學），沒有再教化學。
- 每單元有完整講義：核心觀念、觀念澄清、公式與推導、標準解題流程、快解技巧、30 秒觀念測驗。
- 27 份小考題庫：6 份總複習／學期卷，外加 21 個單元小考；全部為四選一選擇題，交卷即看逐題詳解。
- 「理化模擬測驗」涵蓋全部 21 個單元，約 40 分鐘、全選擇題，是原創練習，不是官方題本節錄。
- 「我的考卷」保存本機瀏覽器考過的卷子，可回看當次題目與詳解。
- 資料來源審核庫：三層資料優先規則（官方資料 > 臺灣出版社 > 經驗證的網路資料），並逐條標示常見說法「通過／有條件／不採用」。

## 與會考數學作戰室的差異（請務必先讀）

- **理化不是獨立會考科目。** 國中教育會考的「自然科」合併生物、理化、地球科學命題，全部為選擇題、無非選擇題，正式題數與時間以官方最新公告為準（近年約 46 題、60 分鐘）。本站只聚焦理化範圍。
- **沒有「十年逐題分析」頁。** 會考數學作戰室的十年分析，是把官方題本逐題人工核對主概念後才能做的統計；本站沒有取得官方自然科逐年 PDF 可供逐題核對理化占比，為避免呈現未經查證的數字，改成「考試結構與領域分布」頁，只呈現官方公開的一般性資訊與本站自己的單元領域分布。
- **沒有逐題轉錄的歷屆考卷。** 歷屆十年頁提供官方原始題本連結（`https://cap.rcpet.edu.tw/exam/<年度>/<年度>exam.html`），但 `archive-exams.js` 目前刻意留空，不假造「官方歷屆試題」內容；之後如取得官方 PDF，可依會考數學作戰室相同的逐題轉錄格式補上。
- **單元順序已對照三版本公開章節目錄，但未逐頁核對課本內文。** 21 個單元的年級／學期／章節順序，是依康軒、翰林、南一三版本公開的章節目錄交叉核對後排定（例如：化學相關單元全部集中在國二下學期教完，國三上、下學期全部是物理），但沒有像會考數學作戰室核對翰林課本內文那樣逐頁核對課文與例題；`questions.js` 中每份小考的 `source` 欄位已註明此限制。

## 使用

1. 保留資料夾內所有檔案與子資料夾，不要只傳送 `index.html`。
2. 直接開啟 `index.html` 即可使用。
3. 建議分享時將整個資料夾壓縮成 ZIP，接收者解壓縮後再使用。

## 開發者檢查

`tools/verify-quizzes.js` 是本網站的自我檢查腳本，驗證單元數、小考題庫題數、選項不重複與模擬測驗涵蓋全部單元：

```bash
node tools/verify-quizzes.js
```

學習進度記錄在每位使用者自己的瀏覽器中，不會跟著資料夾傳給別人。

生成題是原創練習，並非官方題目或預測題；正式規格、範圍與占分以國中教育會考官方網站為準。
