# 國中會考作戰室

公開網址：https://iasome.github.io/taiwan-cap-math/

總覽首頁（`index.html`）串連國中教育會考各科的獨立作戰室網站。每一科都是自成一格、可攜式離線使用的資料夾，複製整個資料夾即可分享，不需要安裝套件或登入。

| 數學 | 可使用 | [`數學會考作戰室/`](數學會考作戰室/README.md) |

會考自然科（地科／理化／生物）與社會科（歷史／地理／公民）各自合併命題、不單獨計分，因此拆分為子範圍練習，架構與規格差異請見各科自己的 README。

## 使用

1. 保留資料夾內所有檔案與子資料夾，不要只傳送單一 `index.html`。
2. 開啟根目錄 `index.html` 進入總覽首頁，或直接開啟想練習的科目資料夾裡的 `index.html`。
3. 建議分享時將整個資料夾壓縮成 ZIP，接收者解壓縮後再使用。

學習進度記錄在每位使用者自己的瀏覽器中，不會跟著資料夾傳給別人。

生成題是原創練習，並非官方題目或預測題；正式規格、範圍與占分以國中教育會考官方網站為準。
