# 會考數學作戰室

公開網址：https://iasome.github.io/taiwan-cap-math/

可攜式離線網站。整個資料夾可複製到隨身碟、雲端硬碟或壓縮後分享，不需要安裝套件或登入。

## 內容

- 依 115 年實際題本規格：25 題選擇題、2 題非選擇題、80 分鐘。
- 模擬考選擇題題型依卷別種子打亂；閱讀題組保持連續，兩題非選擇題保留於第二部分。
- 115 年官方題數與能力比例模考，並以 106–115 年共 275 題的逐題主概念分布校準。
- 逐題詳解、觀念、公式、快解技巧與易錯提醒。
- 公式採臺灣國中課堂慣用符號順序，並以翰林版本資料作第二層對照；分數統一以上下堆疊式呈現。
- 28 單元全冊講義與完成進度（保存在瀏覽器本機）。
- 33 份小考題庫：9 份總複習／學期卷，外加翰林 1上～3下共 24 個章節單元小考。
- 小考頁採年級入口；每個年級分上學期、下學期、總複習三個按鈕。
- 單元小考依範圍深度生成 6 或 8 題；學期／總複習卷維持 12 題，進階題只出現在需要靈活應用的單元。
- 每份小考都有專屬 `quiz-banks/<小考ID>/` 題庫維護資料夾，標明基礎／進階模板來源與隨機規則。
- 「我的考卷」會保存本機瀏覽器考過的卷子，可回看當次題目與詳解。
- 每個翰林章節卡片標註 106–115 年會考主概念題號；十年分析頁可回查每一冊每一章的會考題目分布。
- 106–115 年官方數學題本、答案、試題說明均已存於 `official-data`，可離線開啟。
- 三層資料優先規則：教育部／會考官方資料 > 臺灣出版社 > 經驗證的網路資料。
- 128 條技巧與常見說法審核庫，標示「通過／有條件／不採用」及驗證理由。

## 使用

1. 保留資料夾內所有檔案與子資料夾，不要只傳送 `index.html`。
2. 雙擊 `開啟會考數學作戰室.cmd`；也可直接開啟 `index.html`。
3. 建議分享時將整個資料夾壓縮成 ZIP，接收者解壓縮後再使用。

網站、講義、題庫、詳解與官方 PDF 均可離線使用；「年度總頁」及外部來源網站連結需要網路。

學習進度記錄在每位使用者自己的瀏覽器中，不會跟著資料夾傳給別人。

生成題是原創練習，並非官方題目或預測題；正式規格、答案與勘誤以國中教育會考官方網站為準。
