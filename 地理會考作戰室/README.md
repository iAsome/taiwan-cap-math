# 會考地理作戰室

可攜式離線網站。整個資料夾可複製到隨身碟、雲端硬碟或壓縮後分享，不需要安裝套件或登入。

## 範圍界定（先讀這裡，避免誤會）

國中教育會考沒有獨立的「地理」考科。地理是社會科（歷史／地理／公民）的一部分，社會科全部都是四選一選擇題，沒有非選擇題。因此：

- 講義、小考題庫只涵蓋地理這個學習領域，不含歷史、公民。
- 模擬考採用本站自訂規格（30 題四選一、40 分鐘），**不是**官方獨立考科規格；頁面上會清楚聲明。
- 歷屆十年題目，是從 106–115 年官方「社會科」題本中，逐題篩出屬於地理的題目整理而成，共 98 題；少數依賴地圖、圖表、圖片的題目未收錄，各年度試卷開頭會註明省略原因。

## 內容

- 5 大領域、20 個地理單元全冊講義（觀念、成因、關鍵數據、判斷流程、易錯提醒、單元小測）與完成進度（保存在瀏覽器本機）。
- 25 份小考題庫：20 個單元小考＋5 份領域總複習，全部由 `quiz-taxonomy.js` 的小節 template 生成，模考與小考共用同一份題目來源。
- 每份小考都有專屬 `quiz-banks/<小考ID>/` 題庫維護資料夾，標明小節分類與來源。
- 自訂「地理範圍模擬考」：30 題四選一，40 分鐘，涵蓋全部 20 單元，台灣地理與全球關連兩大領域加抽比重。
- 「我的考卷」會保存本機瀏覽器考過的卷子，可回看當次題目與詳解。
- 106–115 年官方社會科題本、答案、試題說明存於 `official-data`，可離線開啟。
- 十年分析頁、歷屆考卷館：顯示地理題數統計、5 大領域分布、單元回填帳本與年度逐題主概念帳本，資料來自 `archive-exams.js` 已轉錄的 98 題。
- 三層資料優先規則：教育部／會考官方資料 > 臺灣出版社（康軒／南一／翰林）> 經驗證的網路資料。
- 技巧與常見說法審核庫，標示「通過／有條件／不採用」及驗證理由。

## 使用

1. 保留資料夾內所有檔案與子資料夾，不要只傳送 `index.html`。
2. 雙擊 `開啟會考地理作戰室.cmd`；也可直接開啟 `index.html`。
3. 建議分享時將整個資料夾壓縮成 ZIP，接收者解壓縮後再使用。

網站、講義、題庫、詳解與官方 PDF 均可離線使用；外部來源網站連結需要網路。

學習進度記錄在每位使用者自己的瀏覽器中，不會跟著資料夾傳給別人。

生成題是原創練習，並非官方題目或預測題；正式規格、答案與勘誤以國中教育會考官方網站為準。
